# Dev / CSharp

Workspace organizado de projetos C# / .NET.

## Estrutura

`
CSharp/
├── Estudos/                        # Exercicios e aprendizado
│   ├── 01-Fundamentos/             # Logica e exercicios basicos
│   ├── 02-Estruturas-Repeticao/    # For, while, loops
│   ├── 03-Vetores/                 # Arrays e vetores
│   ├── 04-POO/                     # Programacao Orientada a Objetos
│   └── 05-Desafios/                # Maratona e competicoes
└── Projetos/                       # Projetos reais / aplicacoes
    └── TaskManager/
`

## Convencoes

- **Nomes de pastas**: PascalCase
- **Cada projeto**: tem seu proprio .sln e .csproj
- **Novos estudos**: criar dentro da categoria adequada em Estudos/
- **Novos projetos**: criar dentro de Projetos/
